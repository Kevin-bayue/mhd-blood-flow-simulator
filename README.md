# MHD Blood Flow Measurement Simulator

An interactive 3D simulator for a school presentation on magnetohydrodynamics and blood-flow measurement.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually:

```text
http://127.0.0.1:5173/
```

## GitHub Pages

This project includes a GitHub Actions workflow that builds the Vite app and publishes it to GitHub Pages.

After pushing to GitHub:

1. Open the repository on GitHub.
2. Go to `Settings` -> `Pages`.
3. Under `Build and deployment`, select `GitHub Actions`.
4. Wait for the `Deploy to GitHub Pages` workflow to finish.
5. Share the Pages URL with classmates.

## Build

```bash
npm run build
```
